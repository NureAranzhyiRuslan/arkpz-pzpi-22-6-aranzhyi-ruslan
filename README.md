Програмна система для прогнозування погоди

```
IoT:
Cенсори для вимірювання температури, вологості, атмосферного тиску. Збір даних з декількох різних точок, для формування більш точних прогнозів у конкретних регіонах.
Mobile:
Сповіщення про значні погодні зміни, що можуть вплинути на щоденні плани користувача. Можливість перегляду щоденних та щотижневих прогнозів погоди, а також детальної інформації про температуру, вологість на годинній основі.
Backend:
Збирання та обробка даних з IoT-сенсорів, аналіз трендів у погодних умовах. Зберігання історичних погодних даних для покращення довгострокових прогнозів і оцінки змін клімату. Генерація звітів та аналітики, що можна використовувати для планування та попередження наслідків погодних явищ. 
Frontend:
Зручний інтерфейс для відображення погодних умов у реальному часі. Можливість перегляду історичних даних з більш детальною інформацією.
```
